### 如何下载
```
conda create -n 你的环境名 python=3.11
conda activate 你的环境名
pip install lunax
```

### 已有功能
- 数据加载和处理
- EDA分析
- 自动化机器学习建模
- 模型评估和解释
- 数据可视化
